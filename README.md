# Spring Boilerplate

## Description

This is a boilerplate for Spring Boot projects.

## Features

- [x] Spring Boot 3.1.1
- [x] Spring Test
- [x] Spring RestDocs
- [x] Basic Response Format
- [x] Basic Error Handling
- [x] Database Connection (MySQL)
- [x] Flyway
- [ ] Logging
- [ ] Spring Security

## Spring RestDocs

RestDocs 페이지 확인

```http request
GET /docs/index.html
```
